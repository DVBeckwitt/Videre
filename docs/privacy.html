Videre privacy policy

Videre does not collect user data during normal app use. User accounts are linked to the Invidious instance the user chooses, whether it is self-hosted or operated by a third party. The developers of Videre do not provide public Invidious servers.

If a user chooses to submit app feedback, Videre prepares a GitHub issue that can include device details, the user's feedback text, and a screenshot uploaded to Imgur. Users can review this information before submitting the issue.

Videre developers do not own, collect, or share user data outside user-initiated feedback.
